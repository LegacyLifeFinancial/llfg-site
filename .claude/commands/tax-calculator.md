Build 1099 tax estimation tools for LLFG agents. Quarterly estimated payments, deduction tracking, year-end summary, tax bracket visualization.

## What to Build
1. Income tracker pulling from commission data (TEAM_DATA, myDeals)
2. Quarterly estimated tax calculator (federal + state)
3. Common deduction tracker (mileage, home office, phone, marketing)
4. Year-end 1099 summary report
5. Tax bracket visualization chart
6. Payment reminder calendar integration

## Integration Points
- Tax Calculator agent (fin_tax) runs calculations
- Accounting skill provides income data
- Calendar Agent sets quarterly payment reminders
