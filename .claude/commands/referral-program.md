Build agent referral tracking system for the LLFG portal. Unique referral links, referral bonuses, multi-level attribution, payout tracking.

## What to Build
1. Unique referral link per agent (e.g., llfg.us/join?ref=agentId)
2. Referral tracking: who referred whom, signup date, first deal date
3. Referral bonus calculation and payout queue
4. Multi-level attribution (direct + indirect referrals)
5. Referral leaderboard
6. Dashboard card showing referral stats

## Integration Points
- Referral Program agent (mkt_referral) tracks attributions
- Gamification awards XP for successful referrals
- Recruiting Agent links referral to pipeline stage
