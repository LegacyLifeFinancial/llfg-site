Build event management for the LLFG portal: webinars, recruiting events, training sessions. RSVP tracking, reminder emails, attendance analytics.

## What to Build
1. Event creation form (title, date, type, capacity, location/link)
2. RSVP system with confirmation emails
3. Attendance tracking (checked-in vs no-show)
4. Post-event follow-up automation
5. Event calendar integration with ptab-calendar
6. Event analytics (registration rate, attendance rate, lead conversion)

## Integration Points
- Event Marketing agent (mkt_events) manages event lifecycle
- Calendar Agent syncs events to agent calendars
- Email Agent sends invites and reminders
