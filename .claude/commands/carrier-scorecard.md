Rate and compare carriers in the LLFG portal on speed, commission accuracy, underwriting ease, chargeback rate. Data-driven carrier selection for agents.

## What to Build
1. Carrier rating form: speed (1-5), accuracy (1-5), UW ease (1-5), support (1-5)
2. Aggregate scores from all agent ratings
3. Chargeback rate per carrier from deal data
4. Commission accuracy tracking (expected vs actual)
5. Side-by-side carrier comparison view
6. Best carrier recommendation per product type

## Integration Points
- Carrier Scorecard agent (spec_carrierscore) aggregates data
- Carrier Comparison agent provides rankings
- Existing CARRIER_INFO data structure
