Build central integration management for the LLFG portal. Connected services status, sync logs, API health, rate limit monitoring.

## What to Build
1. Integration catalog: available + connected services
2. Connection status dashboard (green/yellow/red per service)
3. Sync logs with timestamp, records synced, errors
4. API health monitoring (latency, error rate, rate limits)
5. One-click connect/disconnect per integration
6. Data flow diagram showing how services connect

## Integration Points
- All integration agents report status here
- Webhook Manager shows hook delivery stats
- API Connector manages authentication tokens
