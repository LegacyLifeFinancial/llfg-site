Build agent performance review system for the LLFG portal. Goal setting, quarterly reviews, 360 feedback, improvement plans, promotion recommendations.

## What to Build
1. Goal setting per agent (SMART goals with measurable targets)
2. Quarterly review form (manager fills out + agent self-review)
3. 360 feedback: peers, managers, self — anonymous aggregation
4. Performance score calculation from: AP, deals, training, activity
5. Improvement plan builder for underperforming agents
6. Promotion readiness assessment tied to PROMOTION_REQUIREMENTS

## Integration Points
- Performance Review agent (hr_review) schedules reviews
- KPI Agent provides performance data
- Team Builder agent uses reviews for promotion recommendations
