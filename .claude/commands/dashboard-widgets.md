Build drag-drop dashboard customization for the LLFG portal. Widget library, custom layouts, personal vs team dashboards, data source picker.

## What to Build
1. Widget library: KPI card, chart, table, activity feed, leaderboard, calendar
2. Drag-drop layout (CSS Grid with resize handles)
3. Widget configuration: data source, time range, chart type
4. Save layout per user in localStorage
5. Personal dashboard vs team dashboard toggle
6. Widget templates for common configurations

## Integration Points
- Overview tab (ptab-overview) becomes customizable
- Chart.js renders widget charts
- All agents can be widget data sources
