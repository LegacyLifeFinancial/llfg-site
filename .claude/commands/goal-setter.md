Build agent goal management for the LLFG portal. SMART goals, progress tracking, milestone celebrations, accountability partners.

## What to Build
1. Goal creation: metric (AP, deals, training), target, deadline
2. Progress bar auto-calculated from portal data
3. Milestone alerts at 25/50/75/100% completion
4. Accountability partner assignment (mutual visibility)
5. Goal templates by role (FA goals vs Manager goals)
6. Gamification XP bonus on goal completion

## Integration Points
- KPI Agent provides progress data
- Gamification awards XP on milestones
- Performance Review includes goal completion rate
